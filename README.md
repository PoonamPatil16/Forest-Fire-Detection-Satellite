# 🔥 Satellite Forest Fire Detection

A computer vision project that detects wildfire hotspots in satellite imagery using Python and OpenCV.

## Features

* Detects wildfire regions in satellite images
* Uses HSV color segmentation
* Identifies fire hotspots with bounding boxes
* Counts total detected fire regions

## Technologies

* Python
* OpenCV
* NumPy
* Satellite imagery analysis

## Run the Project

Run the main script:

```bash
python run.py
```

A file selection window will appear where you can choose a satellite wildfire image to analyze.

The system detects wildfire hotspots and highlights them with bounding boxes.
